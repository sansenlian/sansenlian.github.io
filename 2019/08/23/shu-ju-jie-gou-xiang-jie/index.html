最短路径

1. Dijkstra最短路径

不允许边有负值 O(n2)
如果不要求求出分别到达每个点的最短路径就直接用一个dist[]一维数组就好

求出1号到其他个点的最短距离，如果要求出到达每个点的最短路径，就要再用一个数组path[]来存储路径

一号到其他点的初始距离为：

0 1 12 INF INF INF

可以发现到2的距离最短，那就取2号来作为跳跃点去更新dist[]数组

0 1 1+9 1+3 INF INF

跟新完后继续看下dist[]数组，发现4的距离最短，那么就可以再用4作为跳跃点来做下一步计算

0 1 1+3+4 1+3 1+3+13 1+3+15

然后继续做下一步，直到所有点都作为跳跃点（松弛）才结束

    function Dijkstra(nums) {
        let len = nums.length;
        let dist = [],mark = [];
        // 初始化 dist
        for(let i = 0;i<len;i++)
            dist[i] = nums[0][i];
        // 初始化 mark
        for(let i=0;i<len;i++)
            mark[i]=0;
        mark[0] = 1;
        // 开始Dijkstra
        for(let i=1;i<len;i++){
            let min = Infinity,t;
            // 找到离i最近的点,并且没有被标记过的点
            for(let j= 1;j<len;j++){
                if(mark[j] == 0 && dist[j] < min){
                    min = dist[j];
                    t = j;
                }
            }
            mark[t] = 1;
            // 找到最近的点后就开始开始更新 dist
            for(let k = 0;k<len;k++){
                if(dist[k]>dist[t]+nums[t][k])
                    dist[k] = dist[t]+nums[t][k];
            }
        }
        return dist;
    }
